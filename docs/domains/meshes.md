# Meshes

3D mesh generation and manipulation.

## Primitives

- Box, sphere, cylinder, cone, torus
- Planes, grids
- Icosphere, UV sphere

## Operations

- Subdivision (Catmull-Clark, Loop)
- Extrusion
- Boolean (union, difference, intersection)
- Deformation (bend, twist, taper, lattice)
- Smoothing, decimation

## Attributes

- Position, normal, tangent
- UV coordinates (multiple channels)
- Vertex colors
- Custom attributes

## Generation

- L-systems
- Noise-based displacement
- SDF (signed distance field) meshing
