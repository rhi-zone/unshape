# Textures

Procedural texture generation.

## Noise

- Perlin, Simplex, Worley
- Value noise
- Fractal (fBm, ridged, turbulence)

## Patterns

- Gradient, checkerboard
- Bricks, tiles
- Voronoi cells

## Operations

- Blend, composite
- Warp, distort
- Blur, sharpen
- Threshold, levels

## Color

- HSV/RGB conversion
- Gradient mapping
- Color ramps

## Output

- Albedo, normal maps
- Roughness, metallic
- Height, displacement
