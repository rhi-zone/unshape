# 2D Vector

2D vector graphics generation.

## Primitives

- Rectangle, ellipse, polygon
- Path, bezier curves
- Text (outline)

## Operations

- Boolean (union, difference, intersection)
- Offset, stroke
- Simplify, smooth

## Transforms

- Translate, rotate, scale
- Skew
- Warp along path

## Attributes

- Fill (solid, gradient, pattern)
- Stroke (width, caps, joins)
- Opacity, blend mode

## Export

- SVG
- Mesh triangulation
