# Audio

Audio synthesis and processing.

## Oscillators

- Sine, square, triangle, sawtooth
- Wavetable
- Noise (white, pink, brown)
- FM synthesis

## Filters

- Low-pass, high-pass, band-pass
- State variable filter
- Comb, allpass

## Envelopes

- ADSR
- Multi-stage
- Function-based

## Effects

- Delay, reverb, chorus
- Distortion, bitcrusher
- Compression, limiting

## Modulation

- LFO
- Envelope followers
- Parameter routing
